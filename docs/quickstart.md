# Quickstart

## Prerequisites
- .NET SDK 9
- Docker Desktop or compatible container runtime
- Azure CLI (only for deployment)
- An Azure subscription (only for deployment)

## Run locally
```bash
./scripts/bootstrap.sh
./scripts/run-local.sh
```

## Typical next steps
- Replace the sample order domain with your own domain model.
- Add your own authentication provider.
- Move secrets into a real secret store before production use.
